# 음악 앱 

| 구분 | 기능 | 설명 | 필요한 데이터 (테이블) |
| :--- | :--- | :--- | :--- |
| **회원** | **가입/로그인** | 사용자 계정을 만들고 로그인 | `User` (유저) |
| **음악** | **검색 및 조회** | 가수, 앨범, 노래 정보를 찾아서 봄 | `Artist`, `Album`, `Song` |
| **활동** | **플레이리스트** | 나만의 리스트를 만들고 노래를 담음 | `Playlist`, `Playlist_Song` |
| **활동** | **좋아요** | 마음에 드는 노래에 하트 표시 | `Like` (좋아요) |
